Arduino Timer

Arduino Timer is a collection of Arduino sketch files, python scripts, and documentation to facilitate using an Arduino microcontroller as a time measurement device capable of sub-millisecond accuracy.

The two main areas of documentation are:

  • Using an Arduino to Measure Real Time Events
  • Better Timing Through Calibration

Resources used (copies included):

  • ArduinoSerialCommand
  • TimerOne
  • Audiogen
